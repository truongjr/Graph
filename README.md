+ Trong hàm có phần if (!(xT > 0 && yT > 0 && xT < 1000 && yT < 700)) với xT, yT là toạ độ của trọng số cần đặt, các số 0, 1000, 700 có thể chỉnh sửa sao cho phù hợp với phần làm việc của màn hình.
